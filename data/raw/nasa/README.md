# NASA 3I/ATLAS raw images

Эта директория предназначена для **оригинальных файлов** с изображениями и продуктами наблюдений 3I/ATLAS, загруженных с открытых ресурсов NASA / ESA.

Принципы:

- Сюда складываются *неизменённые* файлы (FITS, PNG, JPEG, MP4 и т.п.).
- Каждому файлу соответствует строка в `docs/nasa_image_manifest.md`
  с полем `local_filename`.
- Любая последующая обработка (кропы, контраст, вычитание фона и т.д.)
  сохраняется уже в `data/processed/` или в отдельных ноутбуках.

Примеры источников:

- NASA image gallery for 3I/ATLAS (официальные изображения и видео).  
- ESA / SOHO / ExoMars TGO материалы, если будут скачаны отдельно.

Задача этой папки — быть «сырым архивом» снимков, без интерпретаций.
